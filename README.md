# EdcDataAnalyzer

Analyzes CSV report from EDC-CZ and makes a pretty table. **Run it at https://ondrejkarlik.github.io/EdcDataAnalyzer/index.html**

![Screenshot](doc/promo.png)


# Building

Requires node.js. Build with `npm run build`.
